# Product Brief: calculator asystem

## Executive Summary

Интерактивный веб-калькулятор цен на IT-услуги с AI-помощником. Инструмент для прозрачного расчёта стоимости разработки при встрече с клиентом или самостоятельного использования на сайте. Заменяет громоздкий Excel на современное веб-решение с возможностью экспорта в PDF.

---

## Core Vision

### Problem Statement

IT-компания использует Excel для расчёта стоимости проектов. При работе с клиентами сложно быстро и прозрачно показать из чего складывается цена, обосновать каждую позицию и дать клиенту понимание за что он платит.

### Problem Impact

- Потеря времени на ручной расчёт в Excel
- Клиент не понимает за что платит → меньше доверия
- Excel не масштабируется и неудобен для презентации
- Нельзя быстро "на берегу" расставить ценники

### Why Existing Solutions Fall Short

- Excel — инструмент для себя, не для продаж
- Нет обоснования цен в реальном времени
- Нет возможности быстро поделиться расчётом (PDF)
- Клиент не может сам посчитать заранее

### Proposed Solution

Веб-страница с двумя блоками:
1. **Калькулятор** — интерактивные чекбоксы по категориям (Лендинги, Боты, CRM и т.д.) с ценами и итогом
2. **AI-помощник** — чат-бот, который помогает подобрать услуги и обосновывает цену

Ключевые функции:
- Открытая ценовая политика (все цены видны)
- Выбор функционала до мелочей
- Скачивание PDF с расчётом
- AI консультирует, но не обязателен

### Key Differentiators

- **Прозрачность** — клиент видит каждую позицию и цену
- **AI-обоснование** — не просто цифры, а объяснение ценности
- **Инструмент продаж** — при клиенте открыл → посчитал → убедил
- **Самообслуживание** — клиент может сам посчитать на сайте

## Target Users

### Primary Users

**1. Mac (Владелец IT-компании)**

- **Контекст:** Встреча с клиентом, ноутбук открыт
- **Цель:** Быстро посчитать и убедительно показать цену
- **Боль:** Excel неудобен, нужно обосновывать каждую позицию
- **Успех:** Клиент понял за что платит, доверяет, соглашается

**2. Потенциальный клиент (на сайте)**

- **Контекст:** Владелец малого/среднего бизнеса, стартапер
- **Уровень:** Не технический, "просто хочу сайт"
- **Поведение:** Сам не знает что нужно, тыкает что понравилось
- **Потребность:** Простые объяснения (tooltips с "?"), помощь в выборе
- **Успех:** Понял что ему реально нужно, увидел ценность, готов к разговору

### Secondary Users

N/A (пока только Mac и его клиенты)

### User Journey

**Mac при встрече с клиентом:**
```
Открыл калькулятор → Выбрал категорию (Лендинг/Бот/CRM)
→ Тыкает опции при клиенте → Клиент видит цену в реальном времени
→ AI объясняет "зачем это нужно" → Скачал PDF → Сделка
```

**Клиент сам на сайте:**
```
Зашёл на сайт → "Хочу сайт, не знаю какой"
→ AI: "Расскажите о бизнесе, помогу подобрать"
→ AI рекомендует опции и объясняет ценность (продаёт!)
→ Клиент видит итог → Понимает за что платит → Оставляет контакт / звонит
```

### UX-требования из discovery

- **Tooltips (?)** на каждой опции — простое объяснение что это и зачем
- **AI-консультант** — не просто отвечает, а продаёт и убеждает
- **Простой язык** — без техно-жаргона для обычных клиентов

## MVP Scope

### Core Features

**1. Калькулятор цен**
- 14 категорий из Excel (LP, ИМ, КС, CRM, Приложения, Боты, IT-разработка, Дизайн, Продвижение и др.)
- Чекбоксы с ценами на каждую опцию
- Коэффициенты/количество где нужно
- Автоматический подсчёт итога
- Система скидок (% от суммы)

**2. UX-элементы**
- Tooltips (?) — простое объяснение каждой опции
- Переключение между категориями
- Итоговая панель с суммой (всегда видна)

**3. AI-помощник (Microsoft Copilot)**
- Чат-бот рядом с калькулятором
- Помогает выбрать услуги, обосновывает цену
- Продаёт и убеждает клиента

**4. PDF-выгрузка**
- Скачать расчёт с выбранными опциями и итогом

**5. Дизайн**
- Стиль asystem.kg (корпоративный минимализм)
- Адаптив под ноутбук

### Out of Scope for MVP

- Аналитика
- Форма заявки / лиды
- Интеграции (WhatsApp, Telegram, CRM)
- Сохранение расчётов / история
- Мультиязычность
